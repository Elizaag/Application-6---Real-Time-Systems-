About this project
    <p> This project was an assignment in which we designed and programmed a Real Time System based off our dream company.
        I based my design off of the designs of Sony Interactive Entertainment (Playstation), the game design of EA, and the 
        game design of Nintendo.
        I used the gyroscope to mimick the Gyroscope sensors in the Dual Sense controller and how nintendo uses the Gyroscope in 
        Zelda Breath of the Wild. I used the LEDs the way the Dual Sense controller uses them for in-games actions,
        statuses, notifications, the mic controls, and how different areas of the game will change the color for Ambient lighting. I used the buttons to mimick the buttons on the Dual Sense controller and how they are used in games.
        The buttons, like on any controller and any game, can be used for interactions, standard game play, and quick time events.
